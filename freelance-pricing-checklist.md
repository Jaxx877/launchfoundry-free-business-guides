# Freelance Project Pricing Checklist

## Build the price floor

- [ ] Add target owner compensation and business overhead.
- [ ] Include software, insurance, professional help, benefits, taxes, and a reasonable reserve for your circumstances.
- [ ] Estimate realistic billable capacity after sales, administration, learning, and downtime.
- [ ] Estimate research, meetings, production, review, quality assurance, handoff, and support.
- [ ] Add risk for uncertainty, coordination, dependencies, urgency, and reserved capacity.

## Write the offer boundary

- [ ] Name each deliverable.
- [ ] State review rounds and who consolidates feedback.
- [ ] Document client inputs and access assumptions.
- [ ] Define exclusions and the change-control process.
- [ ] Show price, schedule, payment terms, and the action that starts work.

## Review after delivery

- [ ] Compare estimated and actual effort.
- [ ] Record unplanned changes and delays.
- [ ] Calculate the actual effective rate and margin.
- [ ] Write the one question or assumption that would improve the next estimate.
